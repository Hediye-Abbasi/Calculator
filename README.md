# Simple Java Swing Calculator

This is a simple calculator project written in Java using Swing.
It was created as a practice project for learning basic GUI programming in Java.

## Features
- Basic operations: addition (+), subtraction (-), multiplication (*), division (/)
- Decimal numbers support
- Clear button (C)
- Displays the selected operator on the screen before calculation
- Simple and student-friendly code

## Technologies Used
- Java
- Java Swing (GUI)

## How to Run
1. Make sure Java is installed on your system.
2. Compile the code: javac T4.java
3. Run the program: java T4.jav


## Notes
- The project is intentionally simple and not optimized.
- It is suitable for beginner and student use.
